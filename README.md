# aether-yaml

Shared yaml processing project

```xml
<project>
    <repositories>
        <!-- Pull GitHub releases as a dependency -->
        <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>
    </repositories>
    
    <dependencies>
        <dependency>
            <groupId>com.github.AetherialMist</groupId>
            <artifactId>aether-yaml</artifactId>
            <version>VERSION</version>
        </dependency>
    </dependencies>
</project>
```
